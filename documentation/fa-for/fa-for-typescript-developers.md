# Fa for Typescript developers

## What Typescript can't do

- [x] Meta-programming
- [x] Type reflection
- [x] Built-in reactivity
- [x] True immutability
- [x] Compile to optimized native binary
- [x] Compile to optimized wasm
- [x] Coloress async functions
- [x] Function composition (for plugins, middlewares, migrations, etc.)
- [x] Fields (for GraphQL-like syntax)
- [x] Native type-safe contexts

Other cool features of Fa:

- [x] Type functions
- [x] Additional collections (Bag, Columnar)
- [x] Aliases
- [x] Extract operator
- [x] Pipe operator
- [x] No "let" vs "const" war
